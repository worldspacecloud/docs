### Notable Changes

<ul class="pl-5 list-disc">
  <li>Fixes a wide range of graphical bugs primarily affecting players using 11th Generation Intel graphics, including screen freezes, texture corruption, incorrect lighting, and flickering.</li>
  <li>Resolves a crash that could occur on some AMD graphics cards when loading a resource pack containing an invalid shader.</li>
</ul>
