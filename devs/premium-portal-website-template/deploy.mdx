Step 1: Upload the template to GitHub as a repository

Step 2: Create a file in the repository named: "/wrangler.jsonc" with the content:

```
{
  "name": "vite-site",
  "compatibility_date": "2025-12-05",
  "assets": {
    "directory": "./dist"
  }
}
```

Step 3: Deploy the repository into CloudFlare Workers
